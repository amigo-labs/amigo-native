amigo-native

Rust-powered drop-in replacements for popular Node.js packages. Compiled to native addons via NAPI-RS. Prebuilt for every major platform.

$ npm install @amigo-labs/slugify
01 // Packages

The Collection

Five native packages under the @amigo-labs scope. Each one wraps a battle-tested Rust crate and ships prebuilt binaries for 6 platform targets.

Benchmark Data

03 // Package Size

Install Footprint

@amigo-labs packages ship a single prebuilt native binary per platform — no node_modules tree, no postinstall scripts. Packages with compiled Rust code may have a larger base binary than a pure-JS equivalent, but carry zero transitive dependencies.
